Detailed Description

A class for performing computation and caching operations for a set of Facets.

Author:
Chris Heng <hengkuanyen@gmail.com>
Based on the fSphinx Python library by Alex Ksikes <alex.ksikes@gmail.com>

Constructor & Destructor Documentation

__construct ( $  objects)

Creates a collection of Facets which adds enhancements like batch queries and caching. With batch querying, multiple Facets can be computed with one single call to Sphinx. Facets may be preloaded or cached to improve performance.

Parameters:
mixed$objectsFacet objects to attach. Also handles Sphinx client and data source.
