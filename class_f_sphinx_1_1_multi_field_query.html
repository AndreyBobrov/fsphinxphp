Detailed Description

A class that provides multi-field query object functionality for Sphinx.

Creates a multi-field query object that extracts a list of query terms from a regular query string and allows the user to refine and toggle those terms.

The user_sph_map specifies the mapping between user-facing query field names and their corresponding Sphinx field names. This information is required by Facets for computation.

FSphinx clients that make use of queries with field terms must have extended2 match mode:

$cl->SetMatchMode(SPH_MATCH_EXTENDED2);

Parameters:
array$user_sph_mapMapping between user fields and actual Sphinx fields.
array$user_attr_mapMapping between user fields and Sphinx multi-value attributes.
